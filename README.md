# YouTube Precise Timestamps

Adds millisecond precision to timestamps shown in the YouTube player.

Mainly meant for use in tandem with [yt-gif](https://github.com/Kori-ts/yt-gif).

## Installation

Install `yt-precise-timestamps.user.js` in [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/).
